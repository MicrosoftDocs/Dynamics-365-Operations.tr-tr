1. <span data-ttu-id="49b16-108">Yeni'ye tıklayın.</span><span class="sxs-lookup"><span data-stu-id="49b16-108">Click New.</span></span>
2. <span data-ttu-id="49b16-109">Toplu işe alma projesi alanına bir değer girin.</span><span class="sxs-lookup"><span data-stu-id="49b16-109">In the Mass hire project field, type a value.</span></span>

5. <span data-ttu-id="49b16-112">Proje bitişi alanına bir tarih girin.</span><span class="sxs-lookup"><span data-stu-id="49b16-112">In the Project end field, enter a date.</span></span>

7. <span data-ttu-id="49b16-114">Evet'i tıklatın.</span><span class="sxs-lookup"><span data-stu-id="49b16-114">Click Yes.</span></span>
8. <span data-ttu-id="49b16-115">Pozisyon oluştur'a tıklayın.</span><span class="sxs-lookup"><span data-stu-id="49b16-115">Click Create positions.</span></span>

    * <span data-ttu-id="49b16-119">Yeni çalışanların Kadrolu mu, yoksa Sözleşmeli mi olacağını belirtin.</span><span class="sxs-lookup"><span data-stu-id="49b16-119">Specify whether the new workers will be Employees or Contractors.</span></span>  
10. <span data-ttu-id="49b16-120">İş alanında, pozisyonları oluşturulacak işi seçmek için açılır menü düğmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="49b16-120">In the Job field, click the drop-down button to select the job to create the positions for.</span></span>

    * <span data-ttu-id="49b16-123">Varsayılan tam zamanlı eşdeğeri, seçilen işten alınır.</span><span class="sxs-lookup"><span data-stu-id="49b16-123">The default full-time equivalent value will come from the selected job.</span></span> <span data-ttu-id="49b16-124">Gerekirse bunu değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="49b16-124">You can change this if needed.</span></span>
