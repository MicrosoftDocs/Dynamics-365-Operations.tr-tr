<span data-ttu-id="65642-104">Bu konuda Ambar yönetiminde bulunan işlevi kullanan ambarlarda kullanılabilen stok yenileme stratejileri açıklanmaktadır.</span><span class="sxs-lookup"><span data-stu-id="65642-104">This topic describes the replenishment strategies that are available for warehouses which use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="65642-105">Bu bilgiler Stok yönetiminde bulunan ambar çözümü için geçerli değildir.</span><span class="sxs-lookup"><span data-stu-id="65642-105">The information doesn't apply to the warehousing solution that is available in Inventory management.</span></span> <span data-ttu-id="65642-106">Aşağıdaki stok yenileme stratejileri bulunmaktadır:</span><span class="sxs-lookup"><span data-stu-id="65642-106">The following replenishment strategies are available:</span></span>

-   <span data-ttu-id="65642-107">**Dalga talep stok yenilemesi** – Bu strateji dalga işi oluşturduğunda stok kullanılamıyorsa giden siparişler veya yükler için stok yenileme işi oluşturur.</span><span class="sxs-lookup"><span data-stu-id="65642-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="65642-108">Örneğin stok yenileme işi bir satış siparişi için gerekli olan miktar dalga işlendiğinde kullanılabilir durumda olmadığında oluşturulabilir.</span><span class="sxs-lookup"><span data-stu-id="65642-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>

## <a name="wave-demand-replenishment"></a><span data-ttu-id="65642-116">Dalga talebi stok yenilemesi</span><span class="sxs-lookup"><span data-stu-id="65642-116">Wave demand replenishment</span></span>
