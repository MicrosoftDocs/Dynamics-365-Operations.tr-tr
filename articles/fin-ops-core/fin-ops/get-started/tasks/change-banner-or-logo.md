# <a name="change-the-banner-or-logo"></a><span data-ttu-id="7cbcc-103">Başlığı veya logoyu değiştirme</span><span class="sxs-lookup"><span data-stu-id="7cbcc-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7cbcc-104">Aşağıdaki yordam, tüzel kişilik için görüntülenen logo ya da başlık resmini güncelleştirmek için sistem yöneticilerinin atabileceği adımları listelemektedir.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="7cbcc-105">Bu yöntemi oluşturmak için kullanılan demo veri şirketi USMF'dir.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="7cbcc-106">**Gezinti bölmesi > Modüller > Kuruluş Yönetimi > Kuruluşlar > Tüzel kişilikler**'e gidin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="7cbcc-107">Başlık sayfasını veya logosunu güncelleştirmek istediğiniz bir tüzel kişiliği, soldaki listeden seçin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="7cbcc-108">Zaten seçiliyse, sonraki adıma geçin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="7cbcc-109">**Pano resmi** sekmesini genişletin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="7cbcc-110">**Değiştir**'i seçin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-110">Select **Change**.</span></span>
    
    <span data-ttu-id="7cbcc-111">Bir başlık resmi için ideal çözünürlük 1920 x 281 pikseldir.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="7cbcc-112">Logo için ideal imge genişliği 350 pikseldir.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="7cbcc-113">**Kaydet**'i seçin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-113">Select **Save**.</span></span>
6. <span data-ttu-id="7cbcc-114">**Gezinti bölmesi > Modüller > Ortak > Ortak > Varsayılan pano**'ya gidin.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="7cbcc-115">Yeni başlık sayfası veya logo resmini panoda görmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="7cbcc-116">Şirket seçiciyi kullanarak, şirketi kapak sayfası resmi yüklediğiniz şirketinize değiştirmeniz gerekebilir.</span><span class="sxs-lookup"><span data-stu-id="7cbcc-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
