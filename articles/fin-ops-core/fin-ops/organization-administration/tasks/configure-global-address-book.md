8. <span data-ttu-id="71691-114">**Müşteri** alanında, bir seçenek seçin.</span><span class="sxs-lookup"><span data-stu-id="71691-114">In the **Customer** field, select an option.</span></span>

10. <span data-ttu-id="71691-116">**Potansiyel** alanında, bir seçenek seçin.</span><span class="sxs-lookup"><span data-stu-id="71691-116">In the **Prospect** field, select an option.</span></span>
