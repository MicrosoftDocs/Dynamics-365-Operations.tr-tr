# <a name="create-a-main-account"></a><span data-ttu-id="05ae5-103">Ana hesap oluşturma</span><span class="sxs-lookup"><span data-stu-id="05ae5-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="05ae5-104">Bu görev kılavuzu, mevcut bir hesap planına ana hesap eklemeyi adım adım açıklar.</span><span class="sxs-lookup"><span data-stu-id="05ae5-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="05ae5-105">Bu kayıtta USMF demo şirketi kullanılmaktadır.</span><span class="sxs-lookup"><span data-stu-id="05ae5-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="05ae5-106">**Gezinti bölmesi > Modüller > Genel muhasebe > Hesap planı > Hesaplar > Ana hesaplar**'a gidin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="05ae5-107">**Yeni**'ye tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-107">Click **New**.</span></span>
3. <span data-ttu-id="05ae5-108">**Ana hesap** alanında bir değer girin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="05ae5-109">**Ad** alanına bir değer yazın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="05ae5-110">**Ana hesap türü** alanında, mali tabloda hesapların bakiyesini ve konumunu en iyi temsil eden türü seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="05ae5-111">Listede, ana hesabın ait olduğu hesap kategorisini seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="05ae5-112">Hesap kategorisi varsayılan mali raporlar ve Power BI panosu içeriği için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="05ae5-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="05ae5-113">Listede, seçili satırdaki bağlantıya tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="05ae5-114">Varsayılan borç veya alacak bakiyesini değiştirin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="05ae5-115">**Varsayılan para birimi** alanında, para birimi listesinden bir değer seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="05ae5-116">Listede, istenen kaydı bulun ve seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="05ae5-117">Listede, seçili satırdaki bağlantıya tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="05ae5-118">**Tüzel kişilik bölümü**'nün genişletilmiş görünümüne geçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="05ae5-119">Bir tüzel kişilik seçmek için **Ekle** öğesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="05ae5-120">Listede tüzel kişiliği seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="05ae5-121">**Ekle** öğesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-121">Click **Add**.</span></span>
15. <span data-ttu-id="05ae5-122">Listede, seçili satırı işaretleyin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="05ae5-123">**Askıda** onay kutusunu işaretleyin veya işaretini kaldırın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="05ae5-124">**Mali raporlama** bölümünü genişletin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="05ae5-125">**Döviz kuru türü** alanında, aramayı açmak için açılır menü düğmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="05ae5-126">Listede, **Hesap için döviz kuru türü**'nü seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="05ae5-127">Listede, seçili satırdaki bağlantıya tıklayın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="05ae5-128">**Para birimi çeviri türü** alanında, hesap için döviz kurlarının hesaplanacağı yöntemi seçin.</span><span class="sxs-lookup"><span data-stu-id="05ae5-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="05ae5-129">Sayfayı kapatın.</span><span class="sxs-lookup"><span data-stu-id="05ae5-129">Close the page.</span></span>
