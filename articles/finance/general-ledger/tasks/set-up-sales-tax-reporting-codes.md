[!include [banner](../../includes/banner.md)]

<span data-ttu-id="bab20-104">Satış vergisi raporlama kodları, satış vergisi raporunda listelenen bir alan numarasına karşılık gelir.</span><span class="sxs-lookup"><span data-stu-id="bab20-104">The Sales tax reporting codes refer to a field number that's listed on a sales tax report.</span></span> <span data-ttu-id="bab20-105">Bunlar ülkeye özgü rapor düzenlerinde kullanılır.</span><span class="sxs-lookup"><span data-stu-id="bab20-105">They are used on country-specific report layouts.</span></span> <span data-ttu-id="bab20-106">Bunlar kod raporuna göre Satış vergisi ödemesinde de kullanılır.</span><span class="sxs-lookup"><span data-stu-id="bab20-106">They're also used on the Sales tax payment by code report.</span></span> <span data-ttu-id="bab20-107">Bu raporda, her bir raporlama kodu için özetlenen bir kapanış dönemine ait satış vergisi tutarları gösterilir.</span><span class="sxs-lookup"><span data-stu-id="bab20-107">That report shows sales tax amounts for a settlement period summarized for each reporting code.</span></span> <span data-ttu-id="bab20-108">Satış vergisi raporlama kodlarını oluşturduktan sonra, **Satış vergisi kodu** sayfasından erişebileceğiniz Rapor kurulumu hızlı sekmelerinde bu kodlara başvurabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="bab20-108">After you create Sales tax reporting codes, you can refer to those codes on the Report setup FastTabs, which you can access from the **Sales tax code** page.</span></span> 

<span data-ttu-id="bab20-109">Bu kayıtta DEMF demo şirketi kullanılmaktadır.</span><span class="sxs-lookup"><span data-stu-id="bab20-109">This recording uses the DEMF demo company.</span></span>

1. <span data-ttu-id="bab20-110">**Gezinti bölmesinde** **Vergi > Kurulum > Satış vergisi > Satış vergisi raporlama kodları**'na gidin.</span><span class="sxs-lookup"><span data-stu-id="bab20-110">In the **Navigation pane**, go to **Tax > Setup > Sales tax > Sales tax reporting codes**.</span></span>
2. <span data-ttu-id="bab20-111">**Yeni**'ye tıklayın.</span><span class="sxs-lookup"><span data-stu-id="bab20-111">Click **New**.</span></span>
3. <span data-ttu-id="bab20-112">Raporlama kodunun ait olduğu rapor düzenini seçin.</span><span class="sxs-lookup"><span data-stu-id="bab20-112">Select the report layout that the reporting code belongs to.</span></span> <span data-ttu-id="bab20-113">Bu düzen, bir satış vergisi kodu için kullanılabilecek raporlama kodlarını filtrelemede kullanılır.</span><span class="sxs-lookup"><span data-stu-id="bab20-113">This layout is used to filter the available reporting codes for a sales tax code.</span></span> <span data-ttu-id="bab20-114">Her satış vergisi kodu, bir rapor düzeni kullanan, bir Satış vergisi dairesine ait bir kapatma dönemine aittir.</span><span class="sxs-lookup"><span data-stu-id="bab20-114">Each sales tax code belongs to a settlement period, which belongs to a Sales tax authority, which uses a report layout.</span></span>  
4. <span data-ttu-id="bab20-115">**Raporlama kodu** alanına bir sayı girin.</span><span class="sxs-lookup"><span data-stu-id="bab20-115">In the **Reporting code** field, enter a number.</span></span>

6. <span data-ttu-id="bab20-117">**Kısa açıklama** alanında, dahili amaçlı bir açıklama girin.</span><span class="sxs-lookup"><span data-stu-id="bab20-117">In the **Brief description** field, enter a description for internal purposes.</span></span>
7. <span data-ttu-id="bab20-118">**Kaydet**'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="bab20-118">Click **Save**.</span></span>
