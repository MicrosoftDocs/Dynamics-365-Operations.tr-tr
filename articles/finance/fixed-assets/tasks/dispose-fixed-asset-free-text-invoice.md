# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="08f2f-103">Serbest metin faturası kullanarak sabit kıymeti elden çıkarma</span><span class="sxs-lookup"><span data-stu-id="08f2f-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="08f2f-104">Bu konu, serbest metin faturası kullanarak bir sabit kıymetin nasıl elden çıkarılacağını açıklar.</span><span class="sxs-lookup"><span data-stu-id="08f2f-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="08f2f-105">Gezinti bölmesinde **Modüller > Alacak hesapları > Faturalar > Tüm serbest metin faturaları**'na gidin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="08f2f-106">**Yeni**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-106">Select **New**.</span></span>
3. <span data-ttu-id="08f2f-107">**Müşteri hesabı** alanında bir değer girin veya seçin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="08f2f-108">Varsayılan **Fatura** tarihini doğrulayın ve gerekirse düzenleyin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="08f2f-109">Kalan varsayılan başlık alanlarını, örneğin **Para birimi** gibi doğrulayın ve gerekirse düzenleyin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="08f2f-110">**Fatura satırları** bölümünde **Açıklama** ve **Ana hesap** alanlarını doğrulayın.</span><span class="sxs-lookup"><span data-stu-id="08f2f-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="08f2f-111">Varsayılan **Satış vergisi grubu** ve **Madde satış vergisi grubu** alanlarını doğrulayın.</span><span class="sxs-lookup"><span data-stu-id="08f2f-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="08f2f-112">Sabit kıymet satışının **Birim fiyatını** veya **Tutarını** girin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="08f2f-113">**Satır ayrıntıları** bölümünü seçin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="08f2f-114">**Sabit kıymet** alanında bir değer girin veya seçin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="08f2f-115">**Naklet**'i seçin.</span><span class="sxs-lookup"><span data-stu-id="08f2f-115">Select **Post**.</span></span>
