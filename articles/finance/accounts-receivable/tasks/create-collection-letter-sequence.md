4. <span data-ttu-id="08a84-110">**Tanım** alanına bir değer girin.</span><span class="sxs-lookup"><span data-stu-id="08a84-110">In the **Description** field, type a value.</span></span>  <span data-ttu-id="08a84-111">Ödeme şartları isteğe bağlıdır.</span><span class="sxs-lookup"><span data-stu-id="08a84-111">The terms of payment is optional.</span></span> <span data-ttu-id="08a84-112">Buraya bir değer girerseniz, tahsilat mektubu ücreti faturasında, müşteriyle depolanan ödeme şartları yerine bu ödeme şartları kullanılır.</span><span class="sxs-lookup"><span data-stu-id="08a84-112">If you enter a value here, the collection letter fee invoice will use these terms of payment instead of the terms of payment stored with the customer.</span></span>  
5. <span data-ttu-id="08a84-113">**Tahsilat mektubu kodu** alanında, göndermek istediğiniz ilk tahsilat mektubunun kodunu seçin.</span><span class="sxs-lookup"><span data-stu-id="08a84-113">In the **Collection letter code** field, select the code for the first collection letter that you want to send.</span></span> <span data-ttu-id="08a84-114">İlk tahsilat mektubu, faturadaki vade tarihine, bu satırdaki Gün alanında mehil süresi için girdiğiniz değere ve bu satıra girdiğiniz diğer bilgilere göre oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="08a84-114">The first collection letter is created according to the due date on the invoice, the value that you enter for the grace period in the Days field on this line, and other information that you enter on this line.</span></span>  
6. <span data-ttu-id="08a84-115">**Tanım** alanına bir değer girin.</span><span class="sxs-lookup"><span data-stu-id="08a84-115">In the **Description** field, type a value.</span></span> <span data-ttu-id="08a84-116">Ücretin para birimi, müşteri para biriminin varsayılan değeri olur.</span><span class="sxs-lookup"><span data-stu-id="08a84-116">The currency for the fee defaults to the customer currency.</span></span> <span data-ttu-id="08a84-117">Bu para birimi kodu, fatura para biriminden farklı olabilir.</span><span class="sxs-lookup"><span data-stu-id="08a84-117">This currency code can be different than the invoice currency.</span></span>

9. <span data-ttu-id="08a84-122">**Tanım** alanına bir değer girin.</span><span class="sxs-lookup"><span data-stu-id="08a84-122">In the **Description** field, type a value.</span></span>
10. <span data-ttu-id="08a84-123">**Ana hesap** alanında, ücretler için kullanılacak gelir hesabını seçin.</span><span class="sxs-lookup"><span data-stu-id="08a84-123">In the **Main account** field, select the revenue account that will be used for fees.</span></span>

12. <span data-ttu-id="08a84-125">**Madde Satış vergisi grubu** alanında, aramayı açmak için açılır menü düğmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="08a84-125">In the **Item sales tax group** field, click the drop down button to open the lookup.</span></span> <span data-ttu-id="08a84-126">Ücret üzerinden satış vergileri hesaplamak gerekiyorsa bir madde satış vergisi grubu seçin.</span><span class="sxs-lookup"><span data-stu-id="08a84-126">Select an item sales tax group if sales taxes must be calculated on the fee.</span></span>  
13. <span data-ttu-id="08a84-127">Listede, seçili satırdaki bağlantıya tıklayın.</span><span class="sxs-lookup"><span data-stu-id="08a84-127">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="08a84-128">**Minimum vadesi geçmiş bakiye** alanında tahsilat mektubu gönderilmeden önce gereken minimum vade tarihi bakiyesini girin.</span><span class="sxs-lookup"><span data-stu-id="08a84-128">In the **Minimum overdue balance** field, enter the minimum overdue balance required before a collection letter is sent.</span></span>

19. <span data-ttu-id="08a84-138">**Ana hesap** alanında istediğiniz değerleri belirtin.</span><span class="sxs-lookup"><span data-stu-id="08a84-138">In the **Main account** field, specify the desired values.</span></span>

22. <span data-ttu-id="08a84-141">Listede, seçili satırdaki bağlantıya tıklayın.</span><span class="sxs-lookup"><span data-stu-id="08a84-141">In the list, click the link in the selected row.</span></span>
23. <span data-ttu-id="08a84-142">**Minimum vadesi geçmiş bakiye** alanına bir sayı girin.</span><span class="sxs-lookup"><span data-stu-id="08a84-142">In the **Minimum overdue balance** field, enter a number.</span></span>
24. <span data-ttu-id="08a84-143">**Günler** alanına bir sayı girin.</span><span class="sxs-lookup"><span data-stu-id="08a84-143">In the **Days** field, enter a number.</span></span>
