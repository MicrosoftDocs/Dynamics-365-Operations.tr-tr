## <a name="set-up-competencies-certificates"></a><span data-ttu-id="69c40-112">Yetkinlikleri ayarlama: Sertifikalar</span><span class="sxs-lookup"><span data-stu-id="69c40-112">Set up competencies: Certificates</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="69c40-117">Yetenekleri ayarlama: Beceriler</span><span class="sxs-lookup"><span data-stu-id="69c40-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="69c40-118">**Çalışanlar** listesi sayfasında, önceki yordamda kullanmış olduğunuz çalışanın hala seçili olduğundan emin olun.</span><span class="sxs-lookup"><span data-stu-id="69c40-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="69c40-119">Ardından Eylem Bölmesinde, **Çalışanlar** sekmesindeki **Yetkinlikler** gurubunda **Beceriler**'i seçin.</span><span class="sxs-lookup"><span data-stu-id="69c40-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>

3. <span data-ttu-id="69c40-121">**Beceri** alanında **Proje yönetimi**'ni seçin.</span><span class="sxs-lookup"><span data-stu-id="69c40-121">In the **Skill** field, select **Project management**.</span></span>

7. <span data-ttu-id="69c40-125">**Kaydet**'i seçip sayfayı kapatın.</span><span class="sxs-lookup"><span data-stu-id="69c40-125">Select **Save**, and then close the page.</span></span>
