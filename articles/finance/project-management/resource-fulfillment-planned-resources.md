[!include [banner](../includes/banner.md)]

<span data-ttu-id="b7939-104">Proje yöneticisi, bir proje için gerekli kaynak rollerini planlayabilir.</span><span class="sxs-lookup"><span data-stu-id="b7939-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="b7939-105">Kaynak yöneticisi **Kaynak karşılama** sayfasında planlanan bu kaynakları istek olarak görür ve gerçek kaynakları atayabilir.</span><span class="sxs-lookup"><span data-stu-id="b7939-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="b7939-106">**Tüm projeler** sayfasında, **XYZ Yükseltme Aşaması 2** projesini seçin.</span><span class="sxs-lookup"><span data-stu-id="b7939-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="b7939-107">**Proje**'yi seçin ve sonra **Düzenle**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="b7939-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="b7939-108">**Proje ekibi ve planlama** sekmesinde **Ekle**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="b7939-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="b7939-109">**Rolleri ekle** iletişim kutusunda **Yazılım geliştiricisi** rolünü seçin.</span><span class="sxs-lookup"><span data-stu-id="b7939-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="b7939-110">**Oluştur**'u seçip proje sayfasını kapatın.</span><span class="sxs-lookup"><span data-stu-id="b7939-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="b7939-111">**Kaynak karşılama** sayfasında, **Yazılım geliştirici 1**'i, **XYZ Yükseltme projesi Aşama 2** projesi için seçin.</span><span class="sxs-lookup"><span data-stu-id="b7939-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="b7939-112">Bir çalışan seçip **Ata** öğesini seçin.</span><span class="sxs-lookup"><span data-stu-id="b7939-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="b7939-113">**XYZ Yükseltme projesi Aşama 2** projesi için **Yazılım geliştirici 1** satırının kaldırıldığından emin olun.</span><span class="sxs-lookup"><span data-stu-id="b7939-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="b7939-114">**Proje ekibi ve planlama** sekmesinde, **XYZ Yükseltme Aşama2** projesi için, önceki adımda seçtiniz çalışanın **Yazılım geliştiricisi** olarak eklendiğini kontrol edin.</span><span class="sxs-lookup"><span data-stu-id="b7939-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
