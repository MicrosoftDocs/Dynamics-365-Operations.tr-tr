<span data-ttu-id="1a9d7-106">Perakende hareketlerini düzenlemek için bir Excel dosyası oluşturduğunuzda dosya bazı varsayılan alanlarla doldurulur.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="1a9d7-107">Oluşturulan Excel dosyasında güncelleştirilmesi gereken bir alan varsayılan olarak görünmüyorsa alanı ekleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="1a9d7-108">Excel çalışma kitabındaki bir çalışma sayfasına alan ekleme</span><span class="sxs-lookup"><span data-stu-id="1a9d7-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="1a9d7-109">Perakende hareketlerini düzenleyebilmeniz amacıyla Excel çalışma kitabına alanlar eklemek için aşağıdaki adımları izleyin.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="1a9d7-110">Excel dosyasını **Ekstreler** sayfasından, **Perakende hareketleri** sayfasından veya **Mağaza mali öğeleri** çalışma alanındaki **Hareket doğrulama hataları** kutucuğundan indirin ve açın.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="1a9d7-111">**Tasarım**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-111">Select **Design**.</span></span>
1. <span data-ttu-id="1a9d7-112">İstediğiniz tablo için kalem simgesini seçin ve ardından var olan alanlar listesinden eklemek istediğiniz alanı bulup seçin.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>

1. <span data-ttu-id="1a9d7-115">Güncelleştirme tamamlandıktan sonra yeni sütunun verilerini almak için **Yenile**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="1a9d7-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>
