1. <span data-ttu-id="2551f-118">**Düzeni Değiştir**'i seçin, oluşturduğunuz ilk düzeni seçin ve sonra **Tamam**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="2551f-118">Select **Change Layout**, select the first layout that you created, and then select **OK**.</span></span> <span data-ttu-id="2551f-119">Modül sırasını not edin.</span><span class="sxs-lookup"><span data-stu-id="2551f-119">Make a note of the module order.</span></span> <span data-ttu-id="2551f-120">Düzenle kaydedilen modül düzeniyle eşleşmesi için bunu değiştirin.</span><span class="sxs-lookup"><span data-stu-id="2551f-120">Change it so that it matches the module order that was saved with the layout.</span></span>
