> [!NOTE]
> <span data-ttu-id="1ceca-106">Varolan bir sayfa için, yalnızca bu sayfanın ana yuva altında en az iki modüle sahip olması durumunda, düzen oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="1ceca-106">You can create layouts for an existing page only if that page has at least two modules under the main slot.</span></span>

1. <span data-ttu-id="1ceca-114">**Katıştırılmış düzene Dönüştür**'ü seçin.</span><span class="sxs-lookup"><span data-stu-id="1ceca-114">Select **Convert to Embedded Layout**.</span></span>

1. <span data-ttu-id="1ceca-118">**Düzeni Değiştir**'i seçin, oluşturduğunuz ilk düzeni seçin ve sonra **Tamam**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="1ceca-118">Select **Change Layout**, select the first layout that you created, and then select **OK**.</span></span> <span data-ttu-id="1ceca-119">Modül sırasını not edin.</span><span class="sxs-lookup"><span data-stu-id="1ceca-119">Make a note of the module order.</span></span> <span data-ttu-id="1ceca-120">Düzenle kaydedilen modül düzeniyle eşleşmesi için bunu değiştirin.</span><span class="sxs-lookup"><span data-stu-id="1ceca-120">Change it so that it matches the module order that was saved with the layout.</span></span>
