# <a name="add-a-favicon"></a><span data-ttu-id="31519-103">Favicon ekleme</span><span class="sxs-lookup"><span data-stu-id="31519-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="31519-104">Bu konu, sitenize bir tercih simgesi ekleneceğini açıklamaktadır.</span><span class="sxs-lookup"><span data-stu-id="31519-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="31519-105">Genel Bakış</span><span class="sxs-lookup"><span data-stu-id="31519-105">Overview</span></span>

<span data-ttu-id="31519-106">Bir tercih edilen simge, Web tarayıcısı sekmesinde, adres çubuğunda, gözatma geçmişinde ve diğer yerlerin yanı sıra yer imleri veya Sık Kullanılanlar'da gösterilen küçük bir grafik dosyasıdır.</span><span class="sxs-lookup"><span data-stu-id="31519-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="31519-107">Markanızı temsil ettiğinden ve yeniden zorlayan ve sitenizi müşterilerinizin ziyaret ettiği diğer sitelerden ayırt etmenize yardımcı olacağından sitenize bir tercih edilen bir simge eklemeniz önerilir.</span><span class="sxs-lookup"><span data-stu-id="31519-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="31519-108">Sitenize çeşitli boyutlarda ve dosya türlerindeki birçok tercih edilebilir simge ekleyebilmenize rağmen bu konu, tek bir tercih edilen simgenin nasıl ekleneceğini göstermektedir.</span><span class="sxs-lookup"><span data-stu-id="31519-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="31519-109">Ancak, aynı işlem ve yer, daha fazla tercih edilen simge eklemek için kullanılır.</span><span class="sxs-lookup"><span data-stu-id="31519-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="31519-110">Sitenizin varlık koleksiyonuna bir tercih simgesi yükleyin</span><span class="sxs-lookup"><span data-stu-id="31519-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="31519-111">Sitenizin varlık koleksiyonuna bir tercih simgesi yüklemek için aşağıdakileri takip edin.</span><span class="sxs-lookup"><span data-stu-id="31519-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="31519-112">Soldaki gezinti bölmesinde **Medya Kitaplığı**'nı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="31519-113">Komut çubuğunda, **Karşıya yükle \> Medya Öğelerini Karşıya Yükle**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="31519-114">Dosya Gezgini penceresinde, karşıya yüklemek istediğiniz site simgesi resim dosyasının konumuna göz atın, dosyayı ve ardından **Aç**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="31519-115">**Medya Öğesini Karşıya Yükle** iletişim kutusunda gerekli başlığı ve alt metni girin.</span><span class="sxs-lookup"><span data-stu-id="31519-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="31519-116">Görüntüleri karşıya yükledikten hemen sonra yayımlamak isterseniz, **Karşıya yükleme sonrasında medyayı yayımla** onay kutusunu seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="31519-117">**Yükledikten sonra medya öğelerini yayımla** onay kutusunu seçmezseniz, **Medya öğeleri** sayfasına dönmeniz ve daha sonra site simgesini el ile yayımlamanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="31519-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="31519-118">**Tamam**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-118">Select **OK**.</span></span>
1. <span data-ttu-id="31519-119">Sağdaki özellik bölmesinde, tercih edilen simgenin ortak URL'sini kopyalayın.</span><span class="sxs-lookup"><span data-stu-id="31519-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="31519-120">Bu URL'yi daha sonra kullanacaksınız.</span><span class="sxs-lookup"><span data-stu-id="31519-120">You will use this URL later.</span></span>

## <a name="create-the-html-for-your-favicon"></a><span data-ttu-id="31519-121">Site simgeniz için HTML oluşturma</span><span class="sxs-lookup"><span data-stu-id="31519-121">Create the HTML for your favicon</span></span>

<span data-ttu-id="31519-122">Tercih edilen simgenin HTML'ini oluşturmak için aşağıdaki HTML dizesini kullanın.</span><span class="sxs-lookup"><span data-stu-id="31519-122">To create the HTML for the favicon, use the following HTML string.</span></span> <span data-ttu-id="31519-123">**href** özniteliği için, **site\_simgeniz\_için\_genel\_URL**'yi önceden kopyaladığınız genel URL ile değiştirin.</span><span class="sxs-lookup"><span data-stu-id="31519-123">For the **href** attribute, replace **Public\_URL\_for\_your\_favicon** with the public URL that you copied earlier.</span></span>

## <a name="create-a-page-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="31519-124">Site simgesi için meta etiketi içeren bir sayfa parçası oluşturma</span><span class="sxs-lookup"><span data-stu-id="31519-124">Create a page fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="31519-125">Site simgesi için meta etiketi içeren bir sayfa parçası oluşturmak için aşağıdaki adımları seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-125">To create a page fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="31519-126">**Sayfa parçaları**'na gidin ve **Yeni**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-126">Go to **Page fragments**, and select **New**.</span></span>
1. <span data-ttu-id="31519-127">**Yeni Sayfa Parçası** iletişim kutusunda, sayfa parçasının temel aldığı modül olarak **Meta etiketleri**'ni seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-127">In the **New Page Fragment** dialog box, select **Metatags** as the module that the page fragment is based on.</span></span>
1. <span data-ttu-id="31519-128">Sayfa parçası için bir ad girin ve **Tamam**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-128">Enter a name for the page fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="31519-129">Parça hiyerarşisi ağacında, **Varsayılan meta etiketleri** alt öğesini seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="31519-130">Sağ bölmede, **Meta Etiketleri** altında **Ekle**'yi seçin ve site simgesi için daha önce oluşturduğunuz HTML dizesini girin.</span><span class="sxs-lookup"><span data-stu-id="31519-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="31519-131">**Düzenlemeyi bitir**'i seçin, ardından sayfa parçasını yayımlamak için **Yayımla**'yı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-131">Select **Finish editing**, and then select **Publish** to publish the page fragment.</span></span>

## <a name="add-the-metatag-page-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="31519-132">Meta etiket sayfası parçasını, sayfalarınızın HTML başlık bölümüne ekleyin</span><span class="sxs-lookup"><span data-stu-id="31519-132">Add the metatag page fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="31519-133">Meta etiket sayfası parçasını, sayfalarınızın HTML **başlık** bölümüne eklemek için şu adımları izleyin.</span><span class="sxs-lookup"><span data-stu-id="31519-133">To add the metatag page fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="31519-134">**Şablonlar**a gidin, site simgenizi eklemek istediğiniz sayfaların şablonunu açın ve **Düzenle**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="31519-135">Şablon hiyerarşisi ağacında, **HTML başlığı** kapsayıcısının sağında bulunan üç nokta düğmesini (**...**) seçin ve sonra **Sayfa parçası ekle**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add page fragment**.</span></span>
1. <span data-ttu-id="31519-136">**Sayfa Parçası Seç** iletişim kutusunda daha önce oluşturduğunuz meta etiketi sayfa parçasını ve sonra **Tamam**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-136">In the **Select Page Fragment** dialog box, select the metatag page fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="31519-137">**Düzenlemeyi bitir**'i seçin, ardından şablonu yayımlamak için **Yayımla**'yı seçin.</span><span class="sxs-lookup"><span data-stu-id="31519-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="31519-138">Sitenizde birden fazla şablon kullanılıyorsa, tüm bunlara meta etiketleri sayfa parçasını eklemeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="31519-138">If your site uses more than one template, you must add the metatags page fragment to all of them.</span></span>

<span data-ttu-id="31519-139">Meta etiketleri sayfa parçasını eklediğiniz şablonu temel alan sayfaları önizlerken, tarayıcı sekmesinde site simgesini görmeniz gerekir.</span><span class="sxs-lookup"><span data-stu-id="31519-139">When you preview pages that are based on the template that you added the metatags page fragment to, you should now see the favicon on the browser tab.</span></span>
