# <a name="crop-images"></a><span data-ttu-id="ca388-103">Görüntüleri kırpma</span><span class="sxs-lookup"><span data-stu-id="ca388-103">Crop images</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="ca388-104">Bu konuda, Microsoft Dynamics 365 Commerce' site oluşturucuda görüntüleri kırpma yöntemi açıklanmaktadır.</span><span class="sxs-lookup"><span data-stu-id="ca388-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="ca388-105">Commerce site oluşturucusu Ortam Kitaplığı, farklı modül türleri ve görünüm pencereleri için en iyi duruma getirmek amacıyla görüntüleri kırpmanıza olanak tanır.</span><span class="sxs-lookup"><span data-stu-id="ca388-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="ca388-106">Görüntüyü kırpma</span><span class="sxs-lookup"><span data-stu-id="ca388-106">Crop an image</span></span>

<span data-ttu-id="ca388-107">Site oluşturucuda bir görüntüyü kırpmak için aşağıdaki adımları izleyin.</span><span class="sxs-lookup"><span data-stu-id="ca388-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="ca388-108">Commerce site oluşturucunun sol gezinti bölmesinde, **Ortam Kitaplığı**'nı seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="ca388-109">Ana pencerede, değiştirmek istediğiniz görüntüyü seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="ca388-110">Komut çubuğunda, **Düzenle** öğesini seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="ca388-111">**Düzenleme Modu**'na girmek için görüntüyü seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="ca388-112">**Düzenleme Modu** altında **Görünümü Modüle göre Düzenle**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="ca388-113">**Modül** açılan menüsünde modül türünü seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="ca388-114">**Görünüm türü** açılan menüsünde görünüm türünü seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="ca388-115">**Yerleşim** açılır menüsünden görüntü yerleşimini seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="ca388-116">**Görünüm penceresi** açılan menüsünde görünüm penceresi boyutunu seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="ca388-117">Görüntü, kırpma bölgesini temsil eden alanla kaplanır.</span><span class="sxs-lookup"><span data-stu-id="ca388-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="ca388-118">Kırpma bölgesini gerektiği gibi taşıyın ve yeniden boyutlandırın.</span><span class="sxs-lookup"><span data-stu-id="ca388-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="ca388-119">En boy oranı otomatik olarak korunur.</span><span class="sxs-lookup"><span data-stu-id="ca388-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="ca388-120">Bitirdiğinizde, komut çubuğunda, **Kaydet**'i seçin ve sonra **Düzenlemeyi bitir**'i seçin.</span><span class="sxs-lookup"><span data-stu-id="ca388-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="ca388-121">Özel kırpma tamamlandıktan sonra, görüntü değişiklikleri neredeyse hemen gerçekleşir.</span><span class="sxs-lookup"><span data-stu-id="ca388-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
