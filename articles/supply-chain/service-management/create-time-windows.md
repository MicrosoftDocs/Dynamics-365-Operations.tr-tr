# <a name="create-time-windows"></a><span data-ttu-id="345a0-103">Zaman aralıkları oluşturma</span><span class="sxs-lookup"><span data-stu-id="345a0-103">Create time windows</span></span>   

[!include [banner](../includes/banner.md)]

<span data-ttu-id="345a0-104">Servis siparişi satırlarının planlamasını en iyi duruma getirmek için zaman aralıkları kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="345a0-104">You can use time windows to optimize the scheduling of service order lines.</span></span>

1. <span data-ttu-id="345a0-105">**Servis yönetimi** \> **Kurulum** \> **Servis sözleşmeleri** \> **Zamana aralığı**'na tıklayın.</span><span class="sxs-lookup"><span data-stu-id="345a0-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Time window**.</span></span>
2. <span data-ttu-id="345a0-106">Yeni bir zaman aralığı oluşturmak için **Alt+N** tuşlarına basın.</span><span class="sxs-lookup"><span data-stu-id="345a0-106">Press **Alt+N** to create a new time window.</span></span>
3. <span data-ttu-id="345a0-107">**Zaman aralıkları** alanında zaman aralığı için bir ad girin.</span><span class="sxs-lookup"><span data-stu-id="345a0-107">In the **Time window** field, enter a name for the time window.</span></span>
4. <span data-ttu-id="345a0-108">**Açıklama** alanında zaman aralığı için bir açıklama girin.</span><span class="sxs-lookup"><span data-stu-id="345a0-108">In the **Description** field, enter a description for the time window.</span></span>
5. <span data-ttu-id="345a0-109">**Yöntem** alanında otomatik oluşturulacak yöntemi seçin.</span><span class="sxs-lookup"><span data-stu-id="345a0-109">In the **Method** field, select the method to be autocreated.</span></span> <span data-ttu-id="345a0-110">Yöntem **El ile** türündeyse, servis siparişi satırının hesaplanan tarihten kaç gün önce veya sonra taşınabileceğini belirtin.</span><span class="sxs-lookup"><span data-stu-id="345a0-110">If the method is of the **Manual** type, specify the number of days before or after the calculated date that the service order line can be moved.</span></span>
