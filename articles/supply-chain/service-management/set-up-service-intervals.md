# <a name="set-up-service-intervals"></a><span data-ttu-id="065a2-103">Servis aralıkları ayarlama</span><span class="sxs-lookup"><span data-stu-id="065a2-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="065a2-104">Servis aralığı, servis siparişlerini oluşturduğunuzda servis sözleşmesi satırları için servis siparişi satırlarının oluşturulma sıklığını gösterir.</span><span class="sxs-lookup"><span data-stu-id="065a2-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="065a2-105">**Servis yönetimi** \> **Kurulum** \> **Servis sözleşmeleri** \> **Servis aralıkları**'na tıklayın.</span><span class="sxs-lookup"><span data-stu-id="065a2-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="065a2-106">Yeni bir servis aralığı oluşturun.</span><span class="sxs-lookup"><span data-stu-id="065a2-106">Create a new service interval.</span></span>
3. <span data-ttu-id="065a2-107">Servis aralığının kodunu ve açıklamasını girin.</span><span class="sxs-lookup"><span data-stu-id="065a2-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="065a2-108">**Aralık** alanında aralığı seçin.</span><span class="sxs-lookup"><span data-stu-id="065a2-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="065a2-109">**Sıklık** alanında sıklığı girin.</span><span class="sxs-lookup"><span data-stu-id="065a2-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="065a2-110">Sıklık, bir servis sözleşmesi aralığını elde etmek için aralığı çarpmanız gereken faktördür.</span><span class="sxs-lookup"><span data-stu-id="065a2-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="065a2-111">Servis aralığını kaydetmek için **Alt+S** tuşlarına basın.</span><span class="sxs-lookup"><span data-stu-id="065a2-111">Press **Alt+S** to save the service interval.</span></span>
