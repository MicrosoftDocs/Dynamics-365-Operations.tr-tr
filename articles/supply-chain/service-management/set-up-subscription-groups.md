# <a name="set-up-subscription-groups"></a><span data-ttu-id="cd06e-103">Abonelik gruplarını ayarla</span><span class="sxs-lookup"><span data-stu-id="cd06e-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="cd06e-104">**Hizmet yönetimi** \> **Kurulum** \> **Servis abonelikleri** \> **Abonelik grupları**'na tıklayın.</span><span class="sxs-lookup"><span data-stu-id="cd06e-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="cd06e-105">Yeni abonelik grubu oluşturmak için CTRL+N tuşlarına basın veya **Yeni**'yi tıklatın.</span><span class="sxs-lookup"><span data-stu-id="cd06e-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="cd06e-106">**Abonelik grubu** alanına, yeni abonelik grubu için açıklayıcı bir ad girin.</span><span class="sxs-lookup"><span data-stu-id="cd06e-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="cd06e-107">**Faturalama** alanında, abonelik için faturalama sıklığını seçin.</span><span class="sxs-lookup"><span data-stu-id="cd06e-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="cd06e-108">Faturalama sıklığı, **Dönem türleri** alanında ayarlanan dönem koduyla gösterilir.</span><span class="sxs-lookup"><span data-stu-id="cd06e-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="cd06e-109">Abonelik hareketleri için gelir tahakkuk ettirmek üzere **Gelir tahakkuku** onay kutusunu seçin ve sonra **Tahakkuk** alanında, bir sıklık belirtin.</span><span class="sxs-lookup"><span data-stu-id="cd06e-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="cd06e-110">Tahakkuk sıklığı **Dönemler** formunda ayarlanan bir dönem kodunu temel alır.</span><span class="sxs-lookup"><span data-stu-id="cd06e-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
