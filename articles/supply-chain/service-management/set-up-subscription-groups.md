# <a name="set-up-subscription-groups"></a><span data-ttu-id="90801-103">Abonelik gruplarını ayarla</span><span class="sxs-lookup"><span data-stu-id="90801-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="90801-104">**Hizmet yönetimi** \> **Kurulum** \> **Servis abonelikleri** \> **Abonelik grupları**'na gidin.</span><span class="sxs-lookup"><span data-stu-id="90801-104">Go to **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="90801-105">Yeni bir abonelik grubu oluşturmak için **Yeni**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="90801-105">Select **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="90801-106">**Abonelik grubu** alanına, yeni abonelik grubu için açıklayıcı bir ad girin.</span><span class="sxs-lookup"><span data-stu-id="90801-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

5.  <span data-ttu-id="90801-109">Abonelik hareketleri için gelir tahakkuk ettirmek üzere **Gelir tahakkuku** onay kutusunu seçin ve sonra **Tahakkuk** alanında, bir sıklık belirtin.</span><span class="sxs-lookup"><span data-stu-id="90801-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="90801-110">Tahakkuk sıklığı **Dönemler** formunda ayarlanan bir dönem kodunu temel alır.</span><span class="sxs-lookup"><span data-stu-id="90801-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
