# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="7968a-103">Projeden servis abonelikleri oluşturma</span><span class="sxs-lookup"><span data-stu-id="7968a-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="7968a-104">**Proje yönetimi ve muhasebe** içindeki bir projeden bir servis aboneliği oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="7968a-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="7968a-105">**Proje yönetimi ve muhasebe** \> **Genel** \> **Projeler** \> **Tüm projeler**'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7968a-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="7968a-106">Bir abonelik oluşturmak istediğiniz projeyi açın.</span><span class="sxs-lookup"><span data-stu-id="7968a-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="7968a-107">**Projeler** formunda, **Eylem bölmesinde** **Yönet** sekmesindek **Servis** \> **Servis aboneliği**'ne tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7968a-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="7968a-110">Kalan bilgileri forma girin.</span><span class="sxs-lookup"><span data-stu-id="7968a-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="7968a-111">Artık abonelik için abonelik hareketlerini oluşturabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="7968a-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="7968a-112">Abonelik grubu için numara serisi ayarladıysanız, abonelik grubunu belirttiğinizde abonelik için otomatik olarak bir abonelik tanımlaması oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="7968a-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
