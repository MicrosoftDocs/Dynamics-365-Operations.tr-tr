## <a name="credit-subscription-transactions"></a><span data-ttu-id="28e19-104">Abonelik işlemlerini alacaklandırma</span><span class="sxs-lookup"><span data-stu-id="28e19-104">Credit subscription transactions</span></span>

1.  <span data-ttu-id="28e19-105">**Servis yönetimi** \> **Ortak** \> **Servis abonelikleri** \> **Tüm servis abonelikleri**'ne tıklayın.</span><span class="sxs-lookup"><span data-stu-id="28e19-105">Click **Service management** \> **Common** \> **Service subscriptions** \> **All service subscriptions**.</span></span>

2.  <span data-ttu-id="28e19-106">Bir alacak dekontu oluşturmak istediğiniz abonelik işlemine iliştirilmiş aboneliği seçin.</span><span class="sxs-lookup"><span data-stu-id="28e19-106">Select the subscription attached to the subscription transaction for which you want to create a credit note.</span></span>

3.  <span data-ttu-id="28e19-107">**Analiz** sekmesini seçin ve Eylem Bölmesinde **Ücret hareketleri** düğmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="28e19-107">Select the **Analyze** tab, and then click the **Fee transactions** button on the Action Pane.</span></span>

4.  <span data-ttu-id="28e19-108">**Ücret hareketleri** formundan bir alacak dekontu oluşturmak istediğiniz hareketi seçin.</span><span class="sxs-lookup"><span data-stu-id="28e19-108">From the **Fee transactions** form, select the transaction for which you want to create a credit note.</span></span>

5.  <span data-ttu-id="28e19-109">**İşlevler** \> **Alacak dekontu için seç**'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="28e19-109">Click **Functions** \> **Select for credit note**.</span></span>

6.  <span data-ttu-id="28e19-110">**Alacak dekontu için seç** formundan, alacaklandırmak istediğiniz hareketi seçin ve **Tamam**'a tıklayın.</span><span class="sxs-lookup"><span data-stu-id="28e19-110">From the **Select for credit note** form, select the transaction that you want to credit and then click **OK**.</span></span>


> [!NOTE]
> <P><span data-ttu-id="28e19-111">Alacak dekontunu oluşturduğunuzda, <STRONG>Alacak dekontları</STRONG>'nı seçtiğinizden emin olun.</span><span class="sxs-lookup"><span data-stu-id="28e19-111">When you create the credit note, make sure that you select <STRONG>Credit notes</STRONG>.</span></span> <span data-ttu-id="28e19-112">Bu, <STRONG>Fatura oluştur</STRONG> iletişim kutusundaki <STRONG>Faturalama yöntemi</STRONG> listesinde bulunur.</span><span class="sxs-lookup"><span data-stu-id="28e19-112">This is found in the <STRONG>Invoicing method</STRONG> list in the <STRONG>Create invoice</STRONG> dialog box.</span></span></P>

<span data-ttu-id="28e19-113">**Servis yönetimi parametreleri** formundaki **Alacaklandırma kapsamındaki tahakkuklara ters işlem uygula** alanı **El ile** olarak ayarlanmışsa, hareket için bir alacak dekontu teklifi oluşturmadan önce tahakkuk ettirilen her gelir hareketini tek tek ters kaydetmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="28e19-113">If the **Reverse accruals on crediting** field in the **Service management parameters** form is set to **Manual**, you have to reverse each accrued revenue transaction individually before you create a credit note proposal for the transaction.</span></span>
