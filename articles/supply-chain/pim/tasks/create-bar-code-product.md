# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="5a34d-103">Ürün için barkod oluşturma</span><span class="sxs-lookup"><span data-stu-id="5a34d-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="5a34d-104">Bu konu, örnek olarak M0001 madde numarasını kullanarak el ile nasıl bir barkod oluşturulacağını gösterir.</span><span class="sxs-lookup"><span data-stu-id="5a34d-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="5a34d-105">Bu yöntemi oluşturmak için kullanılan demo veri şirketi USMF'dir.</span><span class="sxs-lookup"><span data-stu-id="5a34d-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="5a34d-106">Girişi sayfasında **Serbest bırakılan ürün bakımı**'nı seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="5a34d-107">**Bağlantılar** bölümünün altındaki **Ürünler > Serbest bırakılan ürünler**'e gidin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="5a34d-108">Listede, istenen kaydı bulun ve seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="5a34d-109">Bu örnek için madde numarası olarak **M0001**'ü seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="5a34d-110">Eylem Bölmesinde, **Stok yönetimi**'ni seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="5a34d-111">**Barkodlar**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="5a34d-112">**Yeni**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-112">Select **New**.</span></span>
7. <span data-ttu-id="5a34d-113">Aşağıdaki listede oluşturulan seçili satırı işaretleyin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="5a34d-114">**Barkod ayarı** alanında bir değer girin veya bir değer seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="5a34d-115">**Barkod** alanına bir değer girin veya seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="5a34d-116">**Barkod** alanına bir değer yazın.</span><span class="sxs-lookup"><span data-stu-id="5a34d-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="5a34d-117">**Miktar** alanına bir sayı girin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="5a34d-118">**Kaydet**'i seçin.</span><span class="sxs-lookup"><span data-stu-id="5a34d-118">Select **Save**.</span></span>
13. <span data-ttu-id="5a34d-119">Sayfayı kapatın.</span><span class="sxs-lookup"><span data-stu-id="5a34d-119">Close the page.</span></span>
