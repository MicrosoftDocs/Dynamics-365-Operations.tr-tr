# <a name="scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="f36fb-103">Planlanan iş emri bakım işleri</span><span class="sxs-lookup"><span data-stu-id="f36fb-103">Scheduled work order maintenance jobs</span></span>

[!include [banner](../../includes/banner.md)]

 

<span data-ttu-id="f36fb-104">**Planlanan iş emri bakım işleri** sayfası, kaynağa tahsis edilen iş emirlerinin genel görünümünü gösterir.</span><span class="sxs-lookup"><span data-stu-id="f36fb-104">The **Scheduled work order maintenance jobs** page shows an overview of the work orders allocated to a resource.</span></span> <span data-ttu-id="f36fb-105">"İnsan kaynakları", "Araç" ve "Makine" kaynak türlerini kullanan iş emirleri gösterilir.</span><span class="sxs-lookup"><span data-stu-id="f36fb-105">Work orders using resource types "Human resources", "Tool", and "Machine" are shown.</span></span> <span data-ttu-id="f36fb-106">Örneğin, sabahleyin hastalık durumunda çağrılan bir bakım çalışanına tahsis edilmiş bir iş emrini hızlı bir şekilde bulmak ve işe hızlıca başka bir çalışanı tahsis etmek için de kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f36fb-106">For example, if a maintenance worker calls in sick, you can use this page to quickly find work orders allocated to the worker, and then allocate another maintenance worker to the job.</span></span>

## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="f36fb-107">Planlamış iş emri bakım işlerini görüntüleme</span><span class="sxs-lookup"><span data-stu-id="f36fb-107">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="f36fb-108">**Varlık yönetimi** > **Ortak** > **İş emirleri** > **Planlanan iş emri bakım işleri**'ne tıklayın.</span><span class="sxs-lookup"><span data-stu-id="f36fb-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="f36fb-109">İş emri yaşam döngüsü durumu "Planlandı" veya "Devam ediyor" olarak ayarlanmış tüm iş emirlerinin listesini görürsünüz.</span><span class="sxs-lookup"><span data-stu-id="f36fb-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="f36fb-110">Listeyi, örneğin bakım çalışanına göre sıralayabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f36fb-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="f36fb-111">Listeyi belirli bir kaynağa veya bakım çalışanına tahsis edilen iş emirlerini gösterecek şekilde sınırlamak için de filtreyi kullanabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f36fb-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="f36fb-112">İş emrindeki notları görebilir ve gerekirse iş emri işini seçip **Notlar**'a tıklayarak yeni notlar ekleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f36fb-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="f36fb-113">Bir iş emrine bir bakım çalışanı tahsis etmek istiyorsanız, iş emrini seçin ve **İş emri**'ne tıklayın.</span><span class="sxs-lookup"><span data-stu-id="f36fb-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>

5. <span data-ttu-id="f36fb-114">**İş emri** sayfası açılır.</span><span class="sxs-lookup"><span data-stu-id="f36fb-114">The **Work order** page opens.</span></span> <span data-ttu-id="f36fb-115">İş emrini belirli bir bakım çalışanına planlamak için **Gönder**'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="f36fb-115">Click **Dispatch** to schedule the work order to a specific maintenance worker.</span></span>
