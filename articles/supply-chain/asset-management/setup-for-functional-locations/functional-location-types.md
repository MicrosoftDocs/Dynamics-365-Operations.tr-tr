## <a name="create-a-default-functional-location-type"></a><span data-ttu-id="c8a5a-111">Varsayılan işlem yapılacak yerleşim türü oluşturma</span><span class="sxs-lookup"><span data-stu-id="c8a5a-111">Create a default functional location type</span></span>

<span data-ttu-id="c8a5a-112">Bu yordam, varsayılan işlem yapılacak yerleşim için kullanılacak varsayılan işlem yapılacak yerleşim türünün nasıl oluşturulacağını gösterir.</span><span class="sxs-lookup"><span data-stu-id="c8a5a-112">This procedure shows how to create a default functional location type to be used for a default functional location.</span></span>

1. <span data-ttu-id="c8a5a-113">**Varlık yönetimi** > **Kurulum** > **İşlem yapılacak yerleşimler** > **İşlem yapılacak yerleşim türleri**'ni seçin.</span><span class="sxs-lookup"><span data-stu-id="c8a5a-113">Select **Asset management** > **Setup** > **Functional locations** > **Functional location types**.</span></span>

3. <span data-ttu-id="c8a5a-115">**İşlem yapılacak yerleşim türü** alanına "Varsayılan" gibi bir işlem yapılacak yerleşim türü kodu girin ve **Ad** alanına adı girin.</span><span class="sxs-lookup"><span data-stu-id="c8a5a-115">Insert a functional location type ID in the **Functional location type** field, for example, "Default", and a name in the **Name** field.</span></span>

<span data-ttu-id="c8a5a-118">Şimdi, yalnızca varsayılan işlem yapılacak yerleşimde kullanılacak varsayılan işlem yapılacak yerleşim türü oluşturulur.</span><span class="sxs-lookup"><span data-stu-id="c8a5a-118">Now, the default functional location type to be used only on a default functional location is created.</span></span> <span data-ttu-id="c8a5a-119">Bu varsayılan işlem yapılacak yerleşim türüne daha fazla gereksinim veya kısıtlama eklemeniz gerekmez.</span><span class="sxs-lookup"><span data-stu-id="c8a5a-119">You should not add any more requirements or restrictions to this default functional location type.</span></span>
