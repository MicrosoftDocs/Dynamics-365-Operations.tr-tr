# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="7a251-103">Günlük girişlerini veya hareketlerini görüntüleme</span><span class="sxs-lookup"><span data-stu-id="7a251-103">View journal entries or transactions</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="7a251-104">Bu yordam, günlük girişlerini veya hareketlerini aramak için Fiş hareketleri sorgusunun nasıl kullanılacağını gösterir.</span><span class="sxs-lookup"><span data-stu-id="7a251-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="7a251-105">General ledger > Inquiries and reports > Voucher transactions (Genel muhasebe > Sorgular ve raporlar > Fiş hareketleri) menüsüne gidin.</span><span class="sxs-lookup"><span data-stu-id="7a251-105">Go to General ledger > Inquiries and reports > Voucher transactions.</span></span>
2. <span data-ttu-id="7a251-106">Filtre ölçütleri tanımlamak istediğiniz alanı seçin.</span><span class="sxs-lookup"><span data-stu-id="7a251-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="7a251-107">Seçili alan için filtre ölçütlerini girin.</span><span class="sxs-lookup"><span data-stu-id="7a251-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="7a251-108">Tek bir değeri veya bir aralığı filtreleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="7a251-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="7a251-109">Bir aralık tanımlarken doğru sözdiziminin kullanıldığından emin olun.</span><span class="sxs-lookup"><span data-stu-id="7a251-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="7a251-110">Değerler çift nokta (..) ile ayrılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="7a251-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="7a251-111">Filtrelemek için kullanacağınız ek tabloları eklemek için Birleştirir sekmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-111">Click the Joins tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="7a251-112">Ağaçta, "Tablolar\Genel yevmiye defteri girişi" öğesini seçin.</span><span class="sxs-lookup"><span data-stu-id="7a251-112">In the tree, select 'Tables\General journal entry'.</span></span>
6. <span data-ttu-id="7a251-113">Birleştirme tablosu ekle'ye tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-113">Click Add table join.</span></span>
7. <span data-ttu-id="7a251-114">Ek bir tablo eklememeye karar verdiyseniz İptal'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-114">Click Cancel if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="7a251-115">Aralık sekmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-115">Click the Range tab.</span></span>
9. <span data-ttu-id="7a251-116">Sorguyu çalıştırmak için Tamam'a tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-116">Click OK to run the query.</span></span>
10. <span data-ttu-id="7a251-117">Hareket matrahı'na tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-117">Click Transaction origin.</span></span>
    * <span data-ttu-id="7a251-118">Fişin seçilen kaydı hakkında ek bilgileri araştırmak için kılavuz hakkında çeşitli düğmeler kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="7a251-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="7a251-119">Hareketin türüne ve özelliklerine bağlı olarak bazı düğmeler kullanılamayabilir.</span><span class="sxs-lookup"><span data-stu-id="7a251-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>  
11. <span data-ttu-id="7a251-120">Sayfayı kapatın.</span><span class="sxs-lookup"><span data-stu-id="7a251-120">Close the page.</span></span>
12. <span data-ttu-id="7a251-121">Orijinal belgeye tıklayın.</span><span class="sxs-lookup"><span data-stu-id="7a251-121">Click Original document.</span></span>
13. <span data-ttu-id="7a251-122">Sayfayı kapatın.</span><span class="sxs-lookup"><span data-stu-id="7a251-122">Close the page.</span></span>
