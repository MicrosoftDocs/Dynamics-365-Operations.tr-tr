[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="a4009-104">Bu yordam, günlük girişlerini veya hareketlerini aramak için Fiş hareketleri sorgusunun nasıl kullanılacağını gösterir.</span><span class="sxs-lookup"><span data-stu-id="a4009-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

2. <span data-ttu-id="a4009-106">Filtre ölçütleri tanımlamak istediğiniz alanı seçin.</span><span class="sxs-lookup"><span data-stu-id="a4009-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="a4009-107">Seçili alan için filtre ölçütlerini girin.</span><span class="sxs-lookup"><span data-stu-id="a4009-107">Enter your filter critieria for the selected field.</span></span>
    * <span data-ttu-id="a4009-108">Tek bir değeri veya bir aralığı filtreleyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="a4009-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="a4009-109">Bir aralık tanımlarken doğru sözdiziminin kullanıldığından emin olun.</span><span class="sxs-lookup"><span data-stu-id="a4009-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="a4009-110">Değerler çift nokta (..) ile ayrılmalıdır.</span><span class="sxs-lookup"><span data-stu-id="a4009-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="a4009-111">Filtrelemek için kullanacağınız ek tabloları eklemek için Birleştirir sekmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="a4009-111">Click the Joins tab to add additional tables from which to filter.</span></span>

8. <span data-ttu-id="a4009-115">Aralık sekmesine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="a4009-115">Click the Range tab.</span></span>
9. <span data-ttu-id="a4009-116">Sorguyu çalıştırmak için Tamam'a tıklayın.</span><span class="sxs-lookup"><span data-stu-id="a4009-116">Click OK to run the query.</span></span>

    * <span data-ttu-id="a4009-118">Fişin seçilen kaydı hakkında ek bilgileri araştırmak için kılavuz hakkında çeşitli düğmeler kullanılabilir.</span><span class="sxs-lookup"><span data-stu-id="a4009-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="a4009-119">Hareketin türüne ve özelliklerine bağlı olarak bazı düğmeler kullanılamayabilir.</span><span class="sxs-lookup"><span data-stu-id="a4009-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>
