# <a name="suspend-leave"></a><span data-ttu-id="f6094-103">İzni askıya al</span><span class="sxs-lookup"><span data-stu-id="f6094-103">Suspend leave</span></span>

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="f6094-104">Seçili izin türleri için izin tahakkuklarının işlenmesini durdurmak üzere bir çalışan için izni askıya alabilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f6094-104">You can suspend leave for an employee to stop leave accruals from being processed for selected leave types.</span></span> 

## <a name="suspend-leave-and-absence-for-an-employee"></a><span data-ttu-id="f6094-105">Çalışan için izin ve devamsızlığı askıya alma</span><span class="sxs-lookup"><span data-stu-id="f6094-105">Suspend leave and absence for an employee</span></span>

1. <span data-ttu-id="f6094-106">Çalışanın kaydında, **izin** seçeneğini belirleyin.</span><span class="sxs-lookup"><span data-stu-id="f6094-106">On the employee's record, select **Leave**.</span></span>

2. <span data-ttu-id="f6094-107">**İzni askıya al**'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="f6094-107">Select **Suspend leave**.</span></span>

3. <span data-ttu-id="f6094-108">**Yeni**'yi seçin.</span><span class="sxs-lookup"><span data-stu-id="f6094-108">Select **New**.</span></span>

4. <span data-ttu-id="f6094-109">**İzin tahakkukunu askıya al** iletişim kutusunda, **İzin türü**'nü ve askıya alma için **Başlangıç tarihi** ve **Bitiş tarihi**'nü seçin.</span><span class="sxs-lookup"><span data-stu-id="f6094-109">In the **Suspend leave accrual** dialog box, select the **Leave type** along with the **Start date** and **End date** for the suspension.</span></span>

<span data-ttu-id="f6094-111">Bir çalışanın izni askıya alındığında tahakkuklar işlenirse, askıya alınmış izin türleri için bir tahakkuk gerçekleştirilmez.</span><span class="sxs-lookup"><span data-stu-id="f6094-111">If accruals are processed while the employee's leave is suspended, no accrual will be made for the suspended leave types.</span></span>
