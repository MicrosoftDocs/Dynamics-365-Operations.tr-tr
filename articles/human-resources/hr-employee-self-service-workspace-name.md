[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="f9a4a-104">Gönüllüler veya personel dışı başka kişiler varsa, **Personel self-servisi** çalışma alanının adını değiştirmek isteyebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f9a4a-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="f9a4a-105">Bu çalışma alanını **Self servis** olarak değiştirebilirsiniz.</span><span class="sxs-lookup"><span data-stu-id="f9a4a-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="f9a4a-106">**Personel self servisi** çalışma alanının adını değiştirmek aynı zamanda Dynamics 365 Human Resources tarafından dahili olarak kullanılan menü öğesini de değiştirir.</span><span class="sxs-lookup"><span data-stu-id="f9a4a-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="f9a4a-107">**HcmEmployeeSelfServiceWorkspace** menu öğesine daha önceden güvenlik özelleştirmeleri uyguladıysanız, eşliği korumak için aynı değişiklikleri **HcmSelfServiceWorkspace** öğesinde de yapmanız gerekir.</span><span class="sxs-lookup"><span data-stu-id="f9a4a-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>
