# <a name="change-the-banner-or-logo"></a><span data-ttu-id="9ec20-103">Başlığı veya logoyu değiştirme</span><span class="sxs-lookup"><span data-stu-id="9ec20-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="9ec20-104">Aşağıdaki yordam, tüzel kişilik için görüntülenen logo ya da başlık resmini güncelleştirmek için sistem yöneticilerinin atabileceği adımları listelemektedir.</span><span class="sxs-lookup"><span data-stu-id="9ec20-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="9ec20-105">Bu yöntemi oluşturmak için kullanılan demo veri şirketi USMF'dir.</span><span class="sxs-lookup"><span data-stu-id="9ec20-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="9ec20-106">Organizasyon yönetimi > Kuruluşlar > Tüzel kişilikler'e gidin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="9ec20-107">Başlık sayfasını veya logosunu güncelleştirmek istediğiniz bir tüzel kişiliği, soldaki listeden seçin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="9ec20-108">Zaten seçiliyse, sonraki adıma geçin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="9ec20-109">Resimler bölümünü genişletin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-109">Expand the Images section.</span></span> <span data-ttu-id="9ec20-110">Zaten genişletilmişse, sonraki adıma geçin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="9ec20-111">Düzenle'yi tıklatın.</span><span class="sxs-lookup"><span data-stu-id="9ec20-111">Click Edit.</span></span>
5. <span data-ttu-id="9ec20-112">Panodaki şirket görüntü türü alanında, 'Başlık'ı seçin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="9ec20-113">Değiştir seçeneğine tıklayın.</span><span class="sxs-lookup"><span data-stu-id="9ec20-113">Click Change.</span></span>
    * <span data-ttu-id="9ec20-114">İpucu: Bir başlık resmi için ideal çözünürlük 1920 x 281 pikseldir.</span><span class="sxs-lookup"><span data-stu-id="9ec20-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="9ec20-115">Logo için ideal imge genişliği 350 pikseldir.</span><span class="sxs-lookup"><span data-stu-id="9ec20-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="9ec20-116">Yükleme tamamlandığında, yeni kapak sayfası veya logo resmi Pano resim bölümünde görmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="9ec20-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="9ec20-117">Kaydet'e tıklayın.</span><span class="sxs-lookup"><span data-stu-id="9ec20-117">Click Save.</span></span>
8. <span data-ttu-id="9ec20-118">Ortak > Ortak > Varsayılan Pano seçeneğine gidin.</span><span class="sxs-lookup"><span data-stu-id="9ec20-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="9ec20-119">Yeni başlık sayfası veya logo resmini panoda görmelisiniz.</span><span class="sxs-lookup"><span data-stu-id="9ec20-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="9ec20-120">Şirket seçiciyi kullanarak, şirketi kapak sayfası resmi yüklediğiniz şirketinize değiştirmeniz gerekebilir.</span><span class="sxs-lookup"><span data-stu-id="9ec20-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
